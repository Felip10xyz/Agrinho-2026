# Agro Forte

Olá

Este pequeno site apresenta alguns assuntos abordando o tema: "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”

O nome do site é "AgroForte", ele fala sobre agricultura sustentável e como nós podemos caminhar para um futuro melhor para o meio ambiente. Nele, há os tópicos: Tecnologias Sustentáveis, Meio Ambiente, Impactos positivos, Dados, uma pequena galeria interativa, um Quiz de 5 perguntas e uma timeline do Agro.

O site não é complexo, ele apenas fornece informações sobre o tema, com pequenas interações.

(O projeto teve o auxílio das Inteligências Artifíciais: ChatGPT, Claude e Gemini e Canva)

Projeto desenvolvido para o concurso Agrinho 2026.
